Sposoby definiowania (określania) kolorów
Sposób Opis Przykład
HEX Określanie z użyciem definicji heksadecymalnych color: red;
RGB Określanie z użyciem definicji RGB color: rgb(255, 87, 51);
Nazwa koloru Określanie z użyciem nazwy kolorów div { color:red; }
RGB + nasycenie koloru Określanie z użyciem definicji RGB oraz nasycenia kolorów div { color: rgb(10%, 40%, 78%); }
